# MoodFood – Mood-Based Food Recommendation Web App

## Overview
MoodFood is a web application that recommends food based on the user's current mood. It provides personalized suggestions, allows users to rate meals, and saves preferences like last selected mood and favorite meals.

## Features
- User authentication (Signup/Login using Local Storage)
- Mood-based food recommendations
- Dynamic UI rendering using JavaScript
- Save last selected mood (state persistence)
- Save favorite meals
- Rating system for recommendations
- Responsive design for mobile and desktop

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Local Storage (for state management)

## How It Works
1. User signs up and logs in
2. Selects a mood (happy, sad, tired, etc.)
3. Gets personalized food suggestions
4. Can rate meals
5. App remembers last selected mood
